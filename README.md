# Ecommerce Store

## Setup Instructions

1. **Clone the repository**:
   ```bash
   git clone https://github.com/y0622157191-code/ecommerce-store.git
   cd ecommerce-store
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the application**:
   ```bash
   npm start
   ```

4. **Access the application**:
   Open your browser and go to `http://localhost:3000`

## Documentation

- This application allows users to browse products, add them to the cart, and make purchases.
- For further documentation, refer to the [Wiki](https://github.com/y0622157191-code/ecommerce-store/wiki).
